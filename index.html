Como publicar:

  1. Faça commit e push de tudo no GitHub.
  2. Vá em **Settings → Pages** no repositório.
  3. Em **Source**, selecione **Branch: main** e **Folder: /docs**.
  4. O GitHub gerará um link `https://.github.io//` apontando para seu `index.html`.
